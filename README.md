# 🎓 ConexaoEdu

API REST desenvolvida durante a disciplina de Desenvolvimento Backend da graduação em Engenharia de Software.

O projeto tem como objetivo gerenciar usuários, cursos e publicações em uma plataforma acadêmica, aplicando conceitos modernos de desenvolvimento back-end utilizando o ecossistema Spring.

---

## 🚀 Tecnologias

- Java 17
- Spring Boot
- Spring Security
- Spring Data JPA
- PostgreSQL
- RabbitMQ
- Docker
- Maven

---

## 📚 Funcionalidades

- Cadastro de usuários
- Autenticação com JWT
- Gerenciamento de cursos
- Gerenciamento de publicações
- Integração com PostgreSQL
- Comunicação assíncrona utilizando RabbitMQ

---
